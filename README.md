# Handwriting Print Studio

**Генератор рукописного текста для Windows и macOS.** Программа превращает
обычный текст из Word, DOCX или TXT в аккуратно оформленные страницы A4,
которые можно сохранить в PDF и распечатать.

[Открыть сайт проекта](https://OlegOg777.github.io/handwriting-print-studio/) |
[Скачать последнюю версию](https://github.com/OlegOg777/handwriting-print-studio/releases)

## Для чего нужна программа

Handwriting Print Studio помогает быстро подготовить:

- личный конспект или памятку;
- учебные карточки и раздаточные материалы;
- прописи и упражнения;
- поздравления и письма;
- заметки для ежедневника;
- макет текста, написанного рукописным шрифтом;
- многостраничный PDF для печати на обычном принтере.

Программа обрабатывает документы локально. Текст не загружается на сторонний
сервер и остаётся на компьютере пользователя.

## Возможности бесплатной версии

- десять готовых кириллических почерков;
- загрузка `.txt` и `.docx`;
- вставка и редактирование текста;
- автоматический перенос строк;
- автоматическое разделение на страницы A4;
- четыре готовых рукописных шрифта и загрузка собственного TTF/OTF;
- настройка размера, полей, интервала и цвета чернил;
- чистый лист, линейка или клетка;
- естественная неровность строк, слов и наклона;
- предпросмотр каждой страницы;
- экспорт многостраничного PDF;
- экспорт отдельных страниц PNG.

## Чем отличается от обычного рукописного шрифта

При обычной печати все строки и буквы располагаются идеально одинаково.
Handwriting Print Studio добавляет небольшие управляемые отклонения:

- строки слегка смещаются;
- слова получают разный наклон;
- расстояния становятся менее механическими;
- каждый вариант можно воспроизвести по его номеру.

Настройки можно уменьшить до нуля, если требуется ровный текст.

## Запуск на Mac

Для простого запуска дважды нажмите `Запустить_на_Mac.command`.

Первый запуск установит необходимые компоненты и может занять несколько минут.
Если macOS заблокирует файл, нажмите по нему правой кнопкой и выберите
`Открыть`.

Ручной запуск:

```bash
cd ~/Desktop/handwriting-print-studio
python3 -m venv .venv
source .venv/bin/activate
python3 -m pip install -r requirements.txt
streamlit run app.py
```

После запуска программа автоматически откроется в браузере. Интернет для
обработки документов не нужен.

## Запуск на Windows

Для простого запуска откройте `Запустить_на_Windows.bat`.

Ручной запуск:

```powershell
cd Desktop\handwriting-print-studio
py -m venv .venv
.venv\Scripts\activate
py -m pip install -r requirements.txt
streamlit run app.py
```

## Шрифты

В основном списке отображаются только десять готовых рукописных шрифтов.
Системные Arial, Times New Roman и другие обычные шрифты не показываются.
Собственный рукописный TTF/OTF можно загрузить через интерфейс.

При первом запуске программа загружает десять бесплатных кириллических шрифтов
из официального репозитория Google Fonts. Они распространяются по лицензии
SIL Open Font License. Копии лицензий сохраняются в `fonts/licenses`.
